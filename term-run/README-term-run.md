## A script to easily develop a tool that needs to call a terminal in Nordix.
- First look at the terminal emulators that Nordix is supposed to use by default 
- If a user has changed this, the script looks for a range of other options 
- This is to be able to develop a tool for Nordix that needs a terminal and then only needs to use "term-run"

- If there is someone who feels like contributing, then all help is welcome, after all, this is to make life with Linux even better for everyone
